RESUMEN

SAPO AI utiliza un modelo Random Forest para clasificacion acustica binaria de disparos. El sistema procesa archivos de audio y video; en el caso de videos, extrae la pista de audio antes de calcular caracteristicas acusticas y aplicar el clasificador.

El modelo actual, SAPO, distingue entre dos clases:

- DISPARO
- NO DISPARO

El modelo limpio fue entrenado evitando data leakage: la division train/test se realiza antes del escalado, y el scaler se ajusta unicamente con los datos de entrenamiento.
